# Visuals Catalogue (PBIX Spec)

This PBIX specification documents the intended dashboard components for the V2 Modernisation track.

## Design language (Plecto/Appian-inspired NOC)
- Dark background
- Rounded KPI tiles with accent strip
- Dense grid (12+ tiles + 6+ visuals per page)
- Donut/pie for composition
- Gauge/speedometer for SLA/health
- Heatmap for hourly patterns
- Map-like scatter for neighbourhood hotspots
- Monitoring console tables for drift & alerts

## Pages (planned)
